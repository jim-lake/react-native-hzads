
# react-native-hzads

## Getting started

`$ yarn add react-native-hzads`


#### iOS

1. pod install

#### Android

1. Add to tour build.gradle in repositories:

```
maven { url 'https://fyber.bintray.com/fairbid-maven' }
```

## Usage

```javascript
import RNHzAds from 'react-native-hzads';

RNHzAds;
```
